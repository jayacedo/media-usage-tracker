# Getting Started

## Installation

1. Download the plugin ZIP file.
2. In WordPress, go to Plugins → Add New.
3. Click Upload Plugin.
4. Select the ZIP file.
5. Activate the plugin.

## Running Your First Scan

1. Open Media Usage Tracker.
2. Click Start Scan.
3. Wait for the scan to finish.

## Reviewing Results

The plugin will show:

- Used media files
- Unused media files
- Duplicate files
- Media quality issues

## Optimization

Review recommendations and safely remove unused media files.
